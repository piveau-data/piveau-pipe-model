## Build and Install
Requirements:
 * Git
 * Maven
 * Java

```bash
$ git clone https://github.com/piveau-data/piveau-pipe-model.git
$ cd piveau-pipe-model
$ mvn install
```

## Use
Add dependency to your project pom file:
```xml
<dependency>
    <groupId>io.piveau</groupId>
    <artifactId>pipe-model</artifactId>
    <version>2.1.1</version>
</dependency>
```
